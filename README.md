# 📊 Sales Analytics Dashboard (Power BI)

## 🔍 Overview
This project analyzes sales data to identify trends, top-performing regions, and high-revenue products. A multi-page interactive dashboard was built using Power BI.

## 🛠️ Tools Used
- Power BI
- Python (Pandas)
- CSV Dataset

## 📈 Features
- KPI Cards (Total Sales, Total Orders)
- Sales Trend Analysis (Monthly)
- Sales by Region
- Top 5 Products
- Category & Segment Analysis
- Interactive Slicers
- Drill-through functionality
